# 📝 MeTasks - Gestão de Tarefas com Colaboração em Tempo Real

## 📝 Descrição do Projeto
O **MeTasks** é uma solução Android que integra produtividade e comunicação direta. O projeto resolve o gap entre o planejamento de tarefas e a execução em grupo, permitindo que usuários discutam pendências, realizem reuniões de *daily* ou prestem suporte técnico imediato através de videoconferência integrada.

Desenvolvido para a disciplina de **Inteligência Artificial**, o projeto utiliza a **Manus AI** para a estruturação lógica e otimização da interface, utilizando o framework open-source **Jitsi Meet** para viabilizar a colaboração visual entre os membros da equipe.

## 🚀 Proposta de Valor (Diferencial)
O diferencial do MeTasks não é apenas listar afazeres, mas sim o **contexto colaborativo**:
*   **Chamadas Diretas por Contexto:** Inicie uma reunião de vídeo a partir de uma tarefa específica.
*   **Foco e Mentoria:** Ideal para desenvolvedores ou estudantes que precisam de auxílio visual imediato para resolver um problema da lista.
*   **Interface Otimizada:** Layout limpo que prioriza a organização, assistido pela inteligência generativa da Manus AI.

## 🛠️ Tecnologias Utilizadas
*   **Desenvolvimento Assistido:** [Manus AI](https://manus.im/) (Estruturação e lógica de interface).
*   **Comunicação (Engine):** Jitsi Meet SDK (WebRTC).
*   **Plataforma:** Android (Java/Kotlin).
*   **Gerenciamento:** Arquitetura otimizada para performance mobile.

## 🔧 Como Executar (Desenvolvedores)


4.  **Execução:** Sincronize o Gradle e execute no emulador ou dispositivo físico.
