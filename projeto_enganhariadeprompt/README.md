# 🚀 Engenharia de Prompt: Atividade Prática
 
## 📝 Descrição do Projeto
Este projeto foi desenvolvido como um desafio prático de Engenharia de Instrução para dominar a comunicação com Modelos de Linguagem (LLMs). O objetivo central foi obter resultados de alta precisão no menor tempo possível, respeitando um limite estrito de apenas 5 iterações por tarefa. 

A dinâmica envolveu um ciclo de refinamento onde cada resposta era analisada criticamente para integrar melhorias no prompt seguinte, conectando erros ao aprendizado geográfico, temporal e de tom.
 
## 🛠️ Tecnologias e Metodologia
* **Modelo de IA:** Qwen.ai.
* **Limite de Tentativas:** 5 prompts por alvo.
* **Processo:** Avaliação crítica da resposta e integração de feedback para o novo prompt.
* **Alvos Definidos:** Uma imagem de astronauta em estilo barroco e um e-mail formal de um pirata para um rei.
 
## 📊 Resultados e Aprendizados
O processo de refinamento permitiu atingir a "perfeição" solicitada pelo professor através do ajuste fino de instruções.
